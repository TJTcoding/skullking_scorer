# Skull King Scorer

A simple web app to track scores for the Skull King card game. Track games of 2-8 players. Supports Skull King and Rascal scoring methods. Choose how many rounds you want to play and customize how many tricks are in each round. Cool charts and game stats at the end. Best Skull King scorer available with more custimization options than any other.
